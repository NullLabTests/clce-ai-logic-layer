# CLCE AI Logic Layer v1.0.0

Exact Python implementation of John F. Sowa’s Common Logic Controlled English (2004)

Python 3.10+    MIT License    NullLabTests/clce-ai-logic-layer

CLCE turns English into exact First-Order Logic (FOL).

Quick Start:
pip install -e .
python -m examples.basic_clce

Examples from Sowa 2004:
Every cat is on a mat.
The woman Sue sees the cat Yojo in the building Grand Central Terminal.
Nothing is an element of the set empty_set.
Bob is a child of Sue.
Declare Grand Central Terminal as name of building.
Declare 'Grand Central Terminal' as name of building.
